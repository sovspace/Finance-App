# Finance App

Android application for managing stock balance

### Key features

* Account with personal information, ability to create new account and update password or avatar
* Save authorization state 
* Revising profile balance with updated prices
* Balance operations - refill, buy, sell
* Save history of operations
* Browsing dynamics of stocks price and machine learning predictions on graph

### Technical requirements:

* Kotlin 1.3
* Target SDK Version - 29
* Minimal SDK Version - 19
* Kotlin coroutines for asynchronous requests 
* Retrofit library to consume REST API server
* Room to locale state saving in SQLite 
* Navigation Components to provide page navigation
* LiveData and Binding
* Material Design components
